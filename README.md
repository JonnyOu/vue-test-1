# Vue+Element-ui实验项目

### 项目描述
```
此项目为学习vue时做的一个小项目，只有一个页面。项目用了element-ui框架，实现了几个网页开发常用的功能
1.顶部导航栏
2.轮播图
3.分页系统
```
